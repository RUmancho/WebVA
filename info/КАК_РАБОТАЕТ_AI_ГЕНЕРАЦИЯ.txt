КАК РАБОТАЕТ AI-ГЕНЕРАЦИЯ ЗАДАНИЙ

Система автоматически генерирует тестовые задания по всем школьным предметам
с использованием искусственного интеллекта (модель deepseek-r1:7b).

═══════════════════════════════════════════════════════════════════════════

ДВА ТИПА ГЕНЕРАТОРОВ:

1. DLL-ГЕНЕРАТОР (для математики)
   - Быстрая генерация за миллисекунды
   - Используется для алгебраических уравнений и неравенств
   - Гарантированно корректные математические задачи

2. AI-ГЕНЕРАТОР (для всех остальных предметов)
   - Используется для 99% тем по всем предметам
   - Автоматически адаптирует стиль под предмет
   - Создаёт разнообразные и качественные вопросы

═══════════════════════════════════════════════════════════════════════════

КАКИЕ ПРЕДМЕТЫ ПОДДЕРЖИВАЮТСЯ:

✓ Алгебра (функции, прогрессии, уравнения)
✓ Геометрия (планиметрия, стереометрия, теоремы)
✓ Физика (механика, термодинамика, электричество, оптика)
✓ Химия (общая, неорганическая, органическая)
✓ Биология (общая биология, ботаника, зоология)
✓ География (физическая, экономическая, география России)
✓ История (древний мир, средние века, новое время)
✓ Обществознание (человек и общество, политика, экономика)
✓ Русский язык (морфология, синтаксис, орфография)
✓ Английский язык (грамматика, лексика, разговорная речь)
✓ Информатика (программирование, данные, сети)

═══════════════════════════════════════════════════════════════════════════

КАК СИСТЕМА РАБОТАЕТ:

1. Учитель выбирает предмет, раздел и тему
2. Устанавливает сложность (лёгкий / средний / хардкор)
3. Выбирает количество вопросов (от 3 до 20)
4. Решает: с вариантами ответов или без

5. Система автоматически:
   - Определяет, какой генератор использовать (DLL или AI)
   - Создаёт промпт с учётом специфики предмета
   - Генерирует вопросы через LLM
   - Проверяет качество и валидность
   - Возвращает готовый тест

═══════════════════════════════════════════════════════════════════════════

ПРИМЕРЫ РАБОТЫ AI:

ФИЗИКА / Механика / Кинематика:
  Вопрос: "Автомобиль движется со скоростью 20 м/с. 
          Какое расстояние он пройдёт за 10 секунд?"
  Варианты: ["100 м", "200 м", "150 м", "250 м"]
  Правильный ответ: "200 м"

ИСТОРИЯ / Древний мир / Древняя Греция:
  Вопрос: "В каком году состоялись первые Олимпийские игры?"
  Варианты: ["776 г. до н.э.", "500 г. до н.э.", "800 г. до н.э.", "1000 г. до н.э."]
  Правильный ответ: "776 г. до н.э."

РУССКИЙ ЯЗЫК / Орфография / Правописание корней:
  Вопрос: "Как правильно написать: выр_щенный?"
  Варианты: ["выращенный", "выросщенный", "вырощенный", "вырасщенный"]
  Правильный ответ: "выращенный"

═══════════════════════════════════════════════════════════════════════════

УРОВНИ СЛОЖНОСТИ:

🟢 ЛЁГКИЙ (difficulty=1)
   - Базовые вопросы на знание определений
   - Простые вычисления
   - Подходит для 5-7 классов

🟡 СРЕДНИЙ (difficulty=2)
   - Стандартный школьный уровень
   - Применение формул и правил
   - Подходит для 8-9 классов

🔴 ХАРДКОР (difficulty=3)
   - Сложные задачи на анализ
   - Комбинированные вопросы
   - Подходит для 10-11 классов и подготовки к ЕГЭ

═══════════════════════════════════════════════════════════════════════════

ТИПЫ ТЕСТОВ:

📝 С ВАРИАНТАМИ ОТВЕТОВ (with_options=True)
   - 4 варианта на выбор (A, B, C, D)
   - Один правильный ответ
   - Быстрая проверка
   - Исключает субъективность

✍️ БЕЗ ВАРИАНТОВ (with_options=False)
   - Ученик вводит ответ самостоятельно
   - Проверяет глубину знаний
   - Автоматическое сравнение с эталоном
   - Система нормализует ответ (убирает лишние пробелы, учитывает разные форматы)

═══════════════════════════════════════════════════════════════════════════

АВТОМАТИЧЕСКАЯ ПРОВЕРКА:

Система умеет:
✓ Сравнивать числовые ответы (3, 3.0, 3.00 - одинаковы)
✓ Игнорировать регистр (Москва = москва)
✓ Убирать лишние пробелы
✓ Распознавать математические обозначения (x=3 = 3 = x = 3)
✓ Сортировать множественные ответы (±2 = 2, -2 = -2, 2)

═══════════════════════════════════════════════════════════════════════════

FALLBACK СИСТЕМА:

Если основной генератор не работает, система автоматически переключается:

  DLL → AI → LOCAL (предопределённые вопросы)

Это гарантирует, что учитель ВСЕГДА получит тест, даже если:
- AI временно недоступен
- DLL библиотека не загружена
- Нет интернета (система работает полностью локально)

Валидация: Автоматическая проверка качества

НАСТРОЙКА OLLAMA:

Если AI-генерация не работает, убедитесь что Ollama установлен:

1. Установите Ollama: https://ollama.ai/
2. Скачайте модель: ollama pull deepseek-r1:7b
3. Запустите: ollama serve
4. Проверьте: ollama run deepseek-r1:7b

Модель займёт ~5GB на диске.
Требования: 8GB RAM, любой процессор (GPU опционально).
