# Скрипты для тестирования

## create_test_users.py

Скрипт для создания тестовых пользователей в базе данных.

### Использование:
```bash
python scripts/create_test_users.py
```

### Создаваемые пользователи:

**Ученики:**
- student1@test.com / student123
- student2@test.com / student123  
- student3@test.com / student123

**Учителя:**
- teacher1@test.com / teacher123
- teacher2@test.com / teacher123
- teacher3@test.com / teacher123
