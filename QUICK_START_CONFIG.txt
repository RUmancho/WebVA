🎯 КАК ИСПОЛЬЗОВАТЬ?
────────────────────────────────────────────────────────────────────────────────

1️⃣ ИЗМЕНИТЬ НАСТРОЙКИ
   
   → Откройте файл: app_config.json
   → Найдите нужную настройку
   → Измените значение
   → Сохраните файл
   → Перезапустите приложение

   Пример:
   {
     "ai_models": {
       "temperature": 0.3,  ← Измените на 0.7 для более креативных ответов
       "default_model": "gpt-4o-mini"  ← Измените на "gpt-4o" для лучшей модели
     }
   }

2️⃣ ОТКЛЮЧИТЬ ЛОГИ ДЛЯ МОДУЛЯ

   → Откройте файл: app_config.json
   → Найдите секцию "logging" → "enabled"
   → Установите false для нужного модуля

   Пример:
   {
     "logging": {
       "enabled": {
         "llm.py": false,  ← Логи llm.py выключены
         "auth.py": true   ← Логи auth.py включены
       }
     }
   }

3️⃣ ИСПОЛЬЗОВАТЬ В КОДЕ

   В любом Python файле:

   from config_manager import get_ai_temperature, get_ai_model, log_info
   
   # Получить настройки
   temperature = get_ai_temperature()  # Из app_config.json
   model = get_ai_model()              # Из app_config.json
   
   # Логирование
   MODULE_NAME = "your_file.py"
   log_info(MODULE_NAME, "Ваше сообщение")

🔥 ПОПУЛЯРНЫЕ НАСТРОЙКИ
────────────────────────────────────────────────────────────────────────────────

📍 В файле app_config.json:

1. Модель AI:
   "ai_models" → "default_model": "gpt-4o-mini"
   Варианты: "gpt-4o", "gpt-4o-mini", "o1-mini", "o1-preview"

2. Temperature (креативность):
   "ai_models" → "temperature": 0.3
   0.0 = точный, 1.0 = сбалансированный, 2.0 = креативный

3. Максимум токенов:
   "ai_models" → "max_tokens": 2000
   Увеличьте для более длинных ответов

4. Логи AI:
   "logging" → "ai_logs": true
   false = отключить все AI логи

5. Логи по модулям:
   "logging" → "enabled" → "llm.py": false
   false = отключить логи конкретного модуля

🎓 ОСНОВНЫЕ ФУНКЦИИ config_manager.py
────────────────────────────────────────────────────────────────────────────────

AI:
  get_ai_model()              - название модели
  get_ai_temperature()        - temperature
  get_ai_max_tokens()         - макс. токенов
  get_available_models()      - список моделей

ЛОГИРОВАНИЕ:
  log_info(module, msg)       - информация
  log_error(module, msg)      - ошибка
  log_warning(module, msg)    - предупреждение
  is_logging_enabled(module)  - проверка статуса

ПРИЛОЖЕНИЕ:
  get_app_title()             - название
  get_database_name()         - имя БД
  get_page_config()           - конфиг Streamlit

УНИВЕРСАЛЬНЫЕ:
  get_config(section, key)    - получить любую настройку
  update_config(s, k, v)      - обновить настройку
