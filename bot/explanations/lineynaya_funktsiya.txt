## Линейная функция

**Линейная функция** — это функция вида **y = kx + b**, где:
- **k** — коэффициент наклона (угловой коэффициент)
- **b** — свободный член (точка пересечения с осью Y)
- **x** — независимая переменная
- **y** — зависимая переменная

### Основные свойства:

1. **График** линейной функции — это **прямая линия**
2. **Область определения**: все действительные числа (x ∈ R)
3. **Область значений**: все действительные числа (y ∈ R)

### Геометрический смысл:

- **k > 0**: функция возрастает (прямая идет вверх слева направо)
- **k < 0**: функция убывает (прямая идет вниз слева направо)
- **k = 0**: функция постоянна (y = b — горизонтальная прямая)
- **b** показывает, где прямая пересекает ось Y (точка (0, b))

### Примеры:

1. **y = 2x + 3**
   - k = 2 (функция возрастает)
   - b = 3 (пересекает ось Y в точке (0, 3))

2. **y = -x + 5**
   - k = -1 (функция убывает)
   - b = 5 (пересекает ось Y в точке (0, 5))

3. **y = 4**
   - k = 0 (постоянная функция)
   - b = 4 (горизонтальная прямая на уровне y = 4)

### Как построить график:

1. Найдите точку пересечения с осью Y: (0, b)
2. Используя коэффициент k, найдите вторую точку:
   - Если k = 2, то при x = 1 получим y = 2·1 + b = 2 + b
3. Проведите прямую через эти две точки

### Применение:

Линейные функции используются для описания:
- Равномерного движения (путь = скорость × время)
- Стоимости товаров (цена = количество × цена за единицу)
- Температурных шкал (перевод между Цельсием и Фаренгейтом)
