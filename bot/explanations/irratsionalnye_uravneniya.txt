## Иррациональные уравнения

**Иррациональные уравнения** — это уравнения, в которых переменная находится под знаком корня (радикала).

### Основные типы иррациональных уравнений:

1. **Уравнения с квадратным корнем:**
   - √(x + 5) = 3
   - √(2x - 1) = x - 2

2. **Уравнения с корнем n-й степени:**
   - ∛(x + 1) = 2
   - ∜(x - 3) = 1

### Метод решения:

**Шаг 1: Изоляция корня**
- Оставьте корень в одной части уравнения
- Перенесите все остальные члены в другую часть

**Шаг 2: Возведение в степень**
- Возведите обе части уравнения в степень, равную показателю корня
- Например, для √(...) возводим в квадрат, для ∛(...) — в куб

**Шаг 3: Решение полученного уравнения**
- Решите обычное алгебраическое уравнение

**Шаг 4: Проверка корней (ОБЯЗАТЕЛЬНО!)**
- Подставьте найденные корни в исходное уравнение
- Отбросьте посторонние корни (которые не удовлетворяют исходному уравнению)

### Пример решения:

**Решить уравнение:** √(x + 5) = 3

**Решение:**

1. Корень уже изолирован: √(x + 5) = 3

2. Возводим обе части в квадрат:
   (√(x + 5))² = 3²
   x + 5 = 9

3. Решаем полученное уравнение:
   x = 9 - 5
   x = 4

4. Проверка:
   √(4 + 5) = √9 = 3 ✓
   Ответ: x = 4

### Важные замечания:

1. **Область определения:**
   - Выражение под корнем должно быть неотрицательным (для квадратного корня)
   - Например, в √(x - 2) должно быть: x - 2 ≥ 0, т.е. x ≥ 2

2. **Посторонние корни:**
   - При возведении в четную степень могут появиться посторонние корни
   - Всегда проверяйте найденные корни!

3. **Множественные корни:**
   - Если в уравнении несколько корней, изолируйте их по одному
   - Или возводите в степень несколько раз

### Типичные ошибки:

❌ Забывают проверить корни
❌ Не учитывают область определения
❌ Неправильно возводят в степень (забывают скобки)
❌ Не изолируют корень перед возведением в степень

### Практические советы:

1. Всегда записывайте область определения перед решением
2. После каждого возведения в степень проверяйте корни
3. Если получается сложное уравнение, попробуйте замену переменной
4. Для уравнений вида √(ax + b) = cx + d сначала изолируйте корень
